# 股票回测系统 - UI 预览

这是一个基于HTML5、CSS3和JavaScript的股票回测系统UI预览，展示了完整的设计方案和交互功能。

## 功能特性

### 📱 响应式设计

- 移动端适配（< 768px）
- 平板适配（768px - 1024px）
- 桌面端适配（> 1024px）

### 🎨 设计系统

- 完整的颜色方案（主色、功能色、中性色）
- 排版系统（字体、字号、字重）
- 间距系统（基础间距、布局间距）
- 阴影系统（4个层级）
- 圆角系统（5种圆角大小）

### 🌓 主题系统

- 浅色模式（默认）
- 深色模式
- 主题切换功能
- 自动保存用户偏好

### 📄 核心页面

1. **仪表盘** - 系统概览和关键指标
2. **数据管理** - 上传和管理股票数据文件
3. **策略管理** - 创建和管理交易策略
4. **回测管理** - 创建和管理回测任务
5. **结果分析** - 回测结果详细分析

### 🧩 组件库

- 按钮（主要、次要、文本、危险）
- 表单（输入框、下拉选择、日期选择器、开关）
- 表格（标准、响应式、排序）
- 卡片（标准、小、大）
- 徽章（主要、成功、警告、错误）
- 进度条
- 通知
- 模态框
- 抽屉
- 工具提示
- 下拉菜单
- 标签页
- 分页
- 步骤条
- 时间轴

### ⚡ 交互功能

- 页面导航
- 侧边栏折叠
- 表单验证
- 实时反馈
- 通知系统
- 加载状态
- 错误处理
- 确认对话框

## 使用方法

### 1. 直接打开

在浏览器中直接打开 `index.html` 文件即可查看预览。

### 2. 使用本地服务器（推荐）

为了获得最佳体验，建议使用本地服务器：

```bash
# 使用 Python
cd ui-preview
python -m http.server 8000

# 使用 Node.js
cd ui-preview
npx http-server -p 8000

# 使用 PHP
cd ui-preview
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 功能演示

### 主题切换

点击右上角的主题切换按钮可以在浅色模式和深色模式之间切换。

### 页面导航

- 顶部导航栏：点击导航链接切换页面
- 侧边栏：点击菜单项切换页面
- 面包屑导航：显示当前页面路径

### 侧边栏折叠

点击侧边栏顶部的折叠按钮可以展开/折叠侧边栏。

### 表单验证

所有表单都包含实时验证功能，输入时会显示错误提示。

### 通知系统

系统包含完整的通知系统，支持成功、警告、错误和信息四种类型的通知。

### 响应式布局

调整浏览器窗口大小可以查看不同屏幕尺寸下的布局效果。

## 自定义和扩展

### 修改颜色主题

在 `css/design-system.css` 中修改CSS变量：

```css
:root {
    --primary-color: #1890ff;
    --success-color: #52c41a;
    --warning-color: #faad14;
    --error-color: #f5222d;
}
```

### 添加新页面

1. 在 `index.html` 中添加新的页面div
2. 在导航中添加对应的链接
3. 在 `js/app.js` 中更新页面名称映射

### 添加新组件

1. 在 `css/components.css` 中添加组件样式
2. 在 `index.html` 中使用组件
3. 在 `js/app.js` 中添加交互逻辑（如果需要）
